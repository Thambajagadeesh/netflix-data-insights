# Netflix Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset.

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Analysis Performed
- Movies vs TV Shows distribution
- Top countries producing Netflix content
- Rating distribution of Netflix titles

## Insights
- Movies dominate the Netflix catalog compared to TV shows
- The United States produces the majority of Netflix content
- TV-MA and TV-14 ratings are the most common

## Dataset
Netflix Movies and TV Shows Dataset
